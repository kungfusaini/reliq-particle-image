# Examples

These demos showcase the enhanced particle image features without mirroring the Reliq Studios site.

Open each HTML file directly in a browser or serve the directory with a static server.

- `basic-demo.html` - Static logo with low-motion settings.
- `advanced-demo.html` - Interactive repulse and click burst behavior.
- `secondary-particles.html` - Secondary layer for depth and atmosphere.
- `animation-demo.html` - Frame-based sprite animation sequence.
- `responsive-demo.html` - Responsive density and size scaling.

Note: Examples load external images with CORS enabled.
